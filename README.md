# 🌙 DreamBond

DreamBond is a relationship simulation game built with Kotlin and Jetpack Compose. Players build emotional connections with virtual companions through conversations, choices, memories, moods, and interactive date scenes.

## ✨ Overview

DreamBond creates a calm, immersive experience inspired by visual novels and relationship simulation games. Players meet unique companions, shape their personalities through choices, unlock memories, and experience story-driven interactions that evolve over time.

Unlike a traditional chatbot, DreamBond focuses on emotional progression, character development, and replayable relationship paths.

## 🎮 Features

### 👥 Multiple Companions

* Mina 🌙 — calm, caring, and emotionally warm
* Alice 🍵 — shy, traditional, and soft-spoken

### 💬 Interactive Dialogue System

* Multiple dialogue choices
* Dynamic responses
* Character-specific conversations

### ❤️ Relationship Progression

* Affection system
* Relationship levels
* Daily progression loop

### 🧠 Memory System

Characters remember:

* Favorite foods
* Favorite dates
* Favorite times of day
* Previous date scenes

### 😊 Mood & Personality System

* Mood changes based on interactions
* Personality shaping through player choices
* Character behavior evolves over time

### 🎭 Date Scenes

* Bingsu Date 🍧
* Cafe Date ☕
* Tea House Date 🍵
* Bookstore Date 📚

### 🔊 Immersive Experience

* Character voice playback
* Background music
* Visual novel inspired presentation
