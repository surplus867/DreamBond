🌙 DreamBond
DreamBond is an original Android app that explores a dream-based companion simulator, where users build emotional connections through interactive conversations and daily progression.


✨ Overview
DreamBond creates a calm, immersive experience where users meet a virtual companion in a dream-like setting. Through dialogue choices and repeated interactions, the relationship evolves over time.


This project focuses on combining:
interactive storytelling
state-driven UI
emotional progression mechanics


📱 Features (MVP)
🧑 Character selection (starting with one companion)
💬 Interactive dialogue system with multiple choices
❤️ Affection system based on user decisions
🌙 “End Day” flow to simulate nightly interactions
🔄 Simple progression loop (day-by-day experience)


🛠 Tech Stack
Kotlin
Jetpack Compose
Navigation Compose
ViewModel + StateFlow
Material 3


🧱 Architecture
MVVM (Model-View-ViewModel)
State-driven UI with Compose
Single-activity architecture
Navigation handled via Navigation Compose
